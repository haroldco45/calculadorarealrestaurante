# CALCULADORA REAL PRO 2026 🚀

**Calculadora Real Pro** es una herramienta avanzada de ingeniería de menú diseñada para el sector gastronómico colombiano. Permite calcular con precisión el costo de materia prima (CMV), gestionar mermas y proyectar el Precio de Venta al Público (PVP) basado en márgenes de utilidad deseados.

## 🛠️ Características Principales

* **Cálculo Dinámico de Mermas:** Controla el desperdicio real de tus insumos para obtener un costeo preciso.
* **Gestión de Márgenes:** Ajusta el porcentaje de utilidad esperado y obtén el PVP sugerido al instante.
* **Análisis Predictivo:** Sistema de retroalimentación en tiempo real que evalúa si tu plato es rentable, está en punto de equilibrio o es una oferta premium.
* **Interfaz Ultra-Moderna:** Construida con **Tailwind CSS** y fuentes **Outfit** para una experiencia de usuario fluida y profesional.
* **Contexto Local:** Incluye un panel de tendencias actualizado para el mercado colombiano y el Bajo Cauca en 2026.

---

## 📐 Especificaciones Técnicas

La calculadora utiliza la fórmula estándar de la industria para el precio de venta basado en el costo:

$$PVP = \frac{Costo\ Total\ Materia\ Prima}{1 - (\% \text{Margen\ deseado} / 100)}$$

### Tecnologías utilizadas:
* **HTML5 / JS (ES6+):** Lógica de cálculo en tiempo real.
* **Tailwind CSS:** Diseño responsivo y moderno.
* **Intl API:** Formateo automático de moneda (COP) y fechas locales.

## 🚀 Instalación y Uso

1. Descarga el archivo `.html`.
2. Ábrelo en cualquier navegador moderno (Chrome, Edge, Safari).
3. Ingresa los insumos de tu receta:
    * **Peso Bruto:** Lo que compraste.
    * **Merma%:** El porcentaje que no se utiliza tras la limpieza/cocción.
    * **Costo Kg:** El precio pagado por kilogramo.
4. Define tu margen objetivo (por defecto 35%) y observa los resultados.
